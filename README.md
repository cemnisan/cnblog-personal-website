# Personal Web Site

## Technologies
- React / Next 
- Strapi (graphQL for image)
- MongoDB with Strapi
- Uptash(Redis) for Comment
- Auth0 for User Authenticate
    ## Libraries
    - Bootstrap
    - MomentJS,Luxon
    - EmailJS
    - Slugify
    - Unfetch
    - React MarkDown

## Features

- You can introduce youself,
- Where you can write your articles,
- A Personal Article Page

## Installation


-React
```sh
    git clone https://github.com/cemnisan/cnblog-personal-website.git
```
.env.example:
```sh
    NEXT_PUBLIC_API_URL = 'YOUR STRAPI LINK'
```
Install dependencies
```sh
    yarn
```
Start the Server
```sh
    yarn dev
```
